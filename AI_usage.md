# AI Usage

This project used **Claude (Anthropic)** as an AI assistant during development.

## How it was used

- Debugging WebSocket connection and polygon rendering issues
- Generating and refining the admin dashboard UI
- Setting up Docker and docker-compose configuration
- Writing MySQL triggers for the audit trail
- Writing the CI/CD pipeline (GitHub Actions)
- Code review and bug fixing throughout development

## What was not AI-generated

- Project idea and system design
- Database schema decisions
- Building polygon coordinates (manually mapped)
- All testing and validation was done manually by the developer

## Note

AI was used as a development tool — similar to using Stack Overflow or documentation. All code was reviewed, understood, and integrated by the developer.
